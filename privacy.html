Privacy Policy

Last updated: April 1, 2026

1. Overview

This application is a personal automation tool for posting videos to TikTok. It does not collect, store, or share any data from third parties. There are no users beyond the owner of the TikTok account.

2. Data We Collect

This application does not collect any personal data from third parties. The only data stored locally are:

  • TikTok API access tokens (stored locally in a .env file on the owner's device)
  • A log of which videos have been posted (stored locally)
  • A counter for caption numbering (stored locally)

None of this data is transmitted to any server other than TikTok's official API endpoints.

3. TikTok API

This application uses TikTok's official Content Posting API. Any data exchanged with TikTok is governed by TikTok's own Privacy Policy: https://www.tiktok.com/legal/privacy-policy.

4. Data Sharing

No data is shared with any third party. This application communicates exclusively with TikTok's official API.

5. Data Security

API credentials are stored locally on the owner's machine and are not transmitted to any third party.

6. Changes to This Policy

This policy may be updated at any time. The latest version will always be available at this URL.

7. Contact

For any questions, contact the owner via TikTok.
